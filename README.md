# Data-Analysis-of-Supermarket-Sales

This project focuses on identifying patterns and predicting sales outcomes for a retail chain across three cities in China. Using a combination of Exploratory Data Analysis (EDA) and Linear Regression, we move from raw data to actionable predictive insights.

Project Overview
- Data Source: Supermarket sales database including product types, quantities, and demographics.
- Key Techniques: Data Preprocessing, Univariate/Bivariate Analysis, and Predictive Modeling.
- Goal: Build a reliable linear regression model to predict sales based on transaction variables.

Process Workflow
- Selection & Cleaning: Identified a high-dimensional dataset and preprocessed it for analysis.
- Exploratory Analysis: Performed univariate and bivariate studies to understand variable distributions.
- Modeling: Took a representative sample to train and validate a Linear Regression model.
- Prediction: Implemented the model to forecast data points based on features like payment method and gender.
